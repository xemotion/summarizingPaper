# Streamlit을 이용하여 웹 페이지 배포하기 

```
pip isntall streamtlit 
```









* conda 설치 시 주의점 
conda + visual studio로 설치해서 진행 
conda 설치 시에 conda init을 해야 시작됨. 

* 가상 환경에서 작업하자 


* streamlit이라는 가상 환경 user 생성 
```
conda create -n streamlit 
```

streamlit 으로 들어감 
``` 
conda activate streamlit 
```
